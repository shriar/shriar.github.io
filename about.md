## Thesis & Projects & Publication

**Undergraduate THESIS – Hydrodynamic Modeling at Meghna Estuary** (Delft3D | Python | GIS) [\[Link\]](https://zenodo.org/records/17403667)
- Developed a coupled wave-tide hydrodynamic model to simulate nearshore hydrodynamics in the Bay of Bengal at the Meghna Estuary using data like Astronomical Constituent, observed Water level and discharge, wind data from ERA5 reanalysis.
- Analyzed water flow patterns and velocity fields under multiple tidal scenarios to evaluate estuarine.

**Project – River Discharge Forecasting Using GNN-GRU Deep Learning** (Python | PyTorch) [\[Link\]](https://github.com/shriar/Discharge-Forcasting-Using-GNN-GRU)
- Designed and trained a hybrid Graph Neural Network (GNN) + Gated Recurrent Unit (GRU) model to predict river discharge from multi-variate hydrometeorological inputs (rainfall, temperature, evaporation) from ERA5 Reanalysis.

**Project – Flood Inundation Analysis: Padma Bridge Impact** (HEC-RAS 2D | Python | GIS)
- Performed 2D hydraulic modeling with a team to compare flood inundation extents before and after Padma Bridge construction; results indicated increased inundated area post-construction.

**Project – Groundwater Potential Zone Mapping – Kishoreganj District** (GIS | GEE | Python) [\[Link\]](https://drive.google.com/file/d/1I9p1s18IaxNT_pLoy4ecArZUjjP9NAYn/view?usp=sharing)
- Used GIS, Python, and Google Earth Engine to delineate groundwater potential zones using AHP method in Kishoreganj with a team to support sustainable irrigation planning for Boro rice cultivation.

**Project – Rainfall-Runoff Modeling – Padma River Basin** (HEC-HMS, Python)
- Built and calibrated a HEC-HMS model in collaboration with a team to assess the rainfall-runoff processes across the Padma River Basin.

**Project – Deep Learning Implementations** (Python | PyTorch)
- Independent implementations: [GPT language model](https://github.com/shriar/GPT), [FIDLAR](https://github.com/shriar/Implementing-FIDLAR-in-Pytorch), and multiple [deep learning paper](https://github.com/shriar/DeepLearning-Paper-Implementation) reproductions — published on GitHub.

**Project – ERA5 Reanalysis Data Processing Tool** (Python) [\[Link\]](https://github.com/shriar/Era5-data-to-excell--for-hydrological-modelling-HEC-HMS--)
- Built a Python pipeline to convert ERA5 gridded reanalysis data into sub-basin time series format compatible with HEC-HMS, streamlining hydrological modeling workflows.

**Project – Delft3D Pre-processing, Post-processing and Visualization Tool** (Python) [\[Link\]](https://github.com/shriar/Delft3d-Pre-and-Post-Processing-and-visualisation-Tool)
- Created some python tools which will help in pre-processing input file into Delft3d model, post-processing the output files from delft3d and help in understanding the output by visualization.

**Poster – Multi-Scale Hydro-meteorological Assessment of Extreme Flooding Events: The Case of Bangladesh's Eastern Deltaic Region** (Delft3D | Python | HEC-HMS) [\[Link\]](https://sciforum.net/manuscripts/26294/slides.pdf)
